# OpenAPI Boilerplate

## Editor

- [Stoplight Studio](https://stoplight.io/)

## Build

### Preparation

- [openapi-ext-tools](https://github.com/t2y/openapi-ext-tools)
- [ReDoc](https://github.com/Redocly/redoc)

```
npm install -g redoc-cli
pip install openapi-ext-tools

cd path/to/project
cd script
pip install -e ./script
```

### Execution

```
./build
```
